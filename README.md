# 🏎️ Mario Kart - Terminal

Projeto desenvolvido em Node.js que simula uma corrida do Mario Kart diretamente pelo terminal.

O jogo é totalmente baseado em lógica, utilizando sorteio, atributos dos personagens e comparação de resultados a cada rodada.

---

## 🎮 Como funciona o jogo

- Dois personagens competem entre si
- A cada rodada, um tipo de pista é sorteado
- Cada personagem possui atributos como:
  - Velocidade
  - Manobrabilidade
  - Poder
- Um dado é rolado e somado ao atributo correspondente
- Quem tiver o maior valor vence a rodada
- No final, é exibido o vencedor da corrida (Empates podem ocorrer)

---

## 🚀 Tecnologias utilizadas

- Node.js
- JavaScript
